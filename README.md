# 🇱🇰 A Multifaceted Analysis of Sri Lankan Payment Adoption, Borrowing Behavior, and Financial Clustering

> A Corporate Capstone Project conducted at the **Department of Statistics & Computer Science, University of Kelaniya**, analyzing financial inclusion in Sri Lanka using statistical analysis and machine learning techniques.

---

## 📖 Project Overview

Financial inclusion plays a vital role in economic development by ensuring that individuals have access to formal financial services. This project provides a comprehensive analysis of financial behavior among Sri Lankan citizens using the **World Bank Global Findex Microdata**.

The study focuses on three major aspects:

- 💳 Digital Payment Adoption
- 💰 Borrowing Behavior
- 📊 Financial Customer Segmentation

Through statistical analysis and machine learning techniques, the project uncovers key factors influencing financial inclusion and provides actionable insights for policymakers and financial institutions.

---

## 🎯 Objectives

- Analyze factors influencing digital payment adoption in Sri Lanka.
- Identify demographic and socioeconomic characteristics affecting borrowing behavior.
- Segment individuals into meaningful financial clusters using unsupervised learning.
- Generate data-driven insights to support financial inclusion initiatives.
